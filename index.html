01. 소개

저는 핀테크, 로보틱스, 엔터프라이즈 분야에 걸친 실무 경험을 가진 풀스택 개발자입니다. Niflers에서 기업 기후 리스크 보고서 20만 건 이상을 자동 생성하는 시스템을 구축했으며, 알고리즘 최적화와 멀티프로세싱을 통해 생성 시간을 10분에서 2분으로 단축했습니다.

백엔드는 Python / FastAPI와 AWS(ECS, Step Functions, Lambda, S3, RDS)를 중심으로 하며, MySQL 스키마 설계부터 React + TypeScript 프론트엔드까지 풀스택에 걸쳐 개발합니다. Dogugonggan에서는 WebRTC와 MQTT를 활용한 로봇 제어 플랫폼 개발을 리드했으며, 현대, SK하이닉스 등의 클라이언트에 서비스를 납품했습니다.
